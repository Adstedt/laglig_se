Implementation notes

  • Remove the chat-history <aside> currently sitting between LeftSidebar and the right column in workspace-shell.tsx.
  • Add a new <aside> inside the content-row (the rounded-tl-xl wrapper), as the leftmost child of main's parent.
  • Content-row gets gap-3 p-3 so both the new card and main sit with a 12px inset from the frame.
  • History card: w-[320px] bg-card border border-border/60 rounded-xl shadow-sm — all four corners rounded.
  • Main loses its p-4 md:p-6 outer padding (the new wrapper provides it); inner pages compensate if needed.
  • Keyboard shortcut + trigger from sidebar still toggles chatHistoryOpen in useLayoutStore — no store change required.
  • Transition: transition-all duration-300 ease-in-out on the new aside; closed state is w-0 mr-0 (collapse to zero with the gap removed) so main reclaims the space smoothly.
