2 — Badge legend

Five derived badges, no schema-stored state

Öppen
closedAt === null AND correctiveActionTask?.completedAt är null. Auditorn jobbar fortfarande på den.
Redo att verifiera
Öppen, men kopplad task är klar. Knappen Markera som åtgärdat öppnar verify-mode-dialog.
Åtgärdad
Stängd via verify-flödet med en verification_note som audit-bevismaterial. Starkast ISO-spår.
Åtgärdad
Stängd direkt utan formell verifiering — auditorn bedömde att den är åtgärdad utan task-spår.
Avskriven
Manuellt stängd med close_reason — anmärkningen blev irrelevant (verksamhet ändrad, krav borttaget, etc.). Genomstrykning + dämpad färg signalerar "avslutad utan verkställande".

Verb sprawl → one verb

User has to learn 3 verbs (Stäng / Verifiera / Markera ändå) that all do roughly "close this finding."

One verb everywhere. Behind the scenes the dialog selects mode (verify / manual-override / simple) based on finding state.
