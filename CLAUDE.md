# Project Instructions

## App Map

`docs/app-map/app-map.json` is a machine-readable map of the entire application (routes, API surface, server actions, cron jobs, data model, lib/ services, AI/RAG pipeline, ingestion, ops gotchas). Read it first when you need to orient in an unfamiliar part of the codebase. `docs/app-map/app-map.html` is the human visual twin. Keep both updated when architecture changes (new route groups, models, crons, pipelines).

## Context7 Usage

When working with libraries, frameworks, or APIs, use Context7 MCP to fetch up-to-date documentation before writing code. This ensures code examples are current and APIs actually exist.
