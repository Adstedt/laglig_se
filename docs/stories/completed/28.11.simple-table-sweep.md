# Story 28.11: Simple-Table Sweep + Tier-0 Conventions

**Epic:** 28 — Unified DataTable Core (`docs/prd/epic-28-unified-datatable-core.md`)
**Status:** Done (2026-07-08)

## Migrated onto the core

- **`cycle-list-table.tsx`** (377 → ~330 lines): shadcn table + hand-rolled
  sort adapter → `<DataTable>` with `useLocalSorting` (created desc default),
  lifecycle-ranked status sort preserved via `accessorFn`, row-click
  navigation via `rowInteraction` (name-cell `<Link>` keeps keyboard /
  open-in-new-tab and is guard-suppressed), cards below 800 (title = name +
  revision type, status badge, laglista/dokument/period metas, skapad
  footer). Filter chips + PageHeader + empty states unchanged.
- **`file-list-view.tsx`** (595 → ~440 lines): the app's last hand-rolled
  sort/resize/selection scaffolding deleted. Now: manual SortingAdapter
  (folders always above files, folder ordering only by name/modified —
  legacy parity), `useLocalStorageColumnState('laglig:files:columns:v1')`
  for clamped resize (legacy `file-list-column-widths` payloads are
  abandoned — widths reset once, accepted), controlled selection translated
  to the parent's toggle/selectAll/clear API, folder open kept as
  DOUBLE-click via `event.detail >= 2` in `onRowClick`, image hover
  thumbnails preserved in the name cell. Public props interface unchanged
  (`documents-browser.tsx` untouched).

## Kept Tier-0 (documented deviations from the plan)

- **`activity-log-table.tsx`**: its day-separator rows (Idag/Igår) live
  INSIDE one table; the core has no in-table group-header render item, and
  `GroupedDataTable` sections would repeat column headers per day — a
  visual regression on a clean 257-line surface.
- **`cycle-items-tab.tsx`**: initially kept for its DOM contracts
  (`data-cycle-item-id` progress jump, `role="presentation"` zones,
  17 pinning unit tests) — **superseded**: migrated onto the core at
  Alexander's request in the same session. The jump anchor moved onto the
  Dokument cell (exists in both renderers), the core's interactive guard
  replaced the stop-propagation zones, and the 6 structural tests were
  rewritten as behavioral ones (19/19 green). Cards below 800 with
  interactive Bedömning/Motivering/Signera on the card face.

Both are recorded with rationale in the new conventions note:
`docs/architecture/table-conventions.md` (Tier-1 vs Tier-0 rules).
