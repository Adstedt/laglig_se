# Story 2.4: Ingest EU Regulations and Directives

## Status
Draft

## Story

**As a** developer,
**I want** to fetch EU regulations and directives in Swedish from EUR-Lex,
**so that** we have comprehensive EU compliance content.

## Acceptance Criteria

1. EUR-Lex API integration
2. Script fetches regulations (3YYYYRNNNN) and directives (3YYYYLNNNN) in Swedish
3. Fields: CELEX, title, full text, publication date, EUT reference
4. Data in `legal_documents` with EU_REGULATION or EU_DIRECTIVE type
5. EU metadata in `eu_documents` table
6. National Implementation Measures (NIM) for directives
7. Cross-references between EU directives â†” Swedish implementing laws
8. Rate limiting per API guidelines
9. Progress logging
10. Complete in <12 hours
11. Verify: 110,000+ EU documents in database
