# Story 14.17: Claude-Style Collapsed Tool Call Display

## Status

Backlog

## Story

**As a** user reviewing an AI assessment or chat response,
**I want** completed tool calls to be collapsed into a single compact summary line,
**so that** the conversation stays focused on the AI's analysis rather than the tool call mechanics.

## Context

Currently each tool call renders as its own row with status icon, label, and detail text. For assessment conversations that typically make 3-5 tool calls (get_company_context, search_laws, get_change_details, suggest_followups), this takes up significant vertical space before the actual assessment content begins.

Claude's desktop client solves this by collapsing completed tool calls into a single line like:
```
✓ Hämtade företagskontext · Sökte i lagdatabasen (2) · Hämtade ändringsdetaljer
```

Running/errored calls stay expanded. The collapsed line is expandable to show individual call details.

## Ideas

- **Collapsed state**: Single row with dot-separated labels for all completed calls. Count in parentheses if same tool called multiple times (e.g., "Sökte i lagdatabasen (3)").
- **Expandable**: Click to expand and see individual rows (current design but tighter)
- **Running calls**: Stay expanded and animated while in progress. Once all complete, collapse.
- **Error calls**: Stay expanded with error styling, don't collapse.
- **Transition**: Animate from expanded (during streaming) to collapsed (after completion) smoothly.
- **Clickable details**: Individual tools within the collapsed view should still be clickable to open sidebar details.
