## What changed

1. **Marketing decoupled first** (the epic's ordering requirement):
   `components/features/landing-v3/marketing-document-table.tsx` is a FROZEN

   interface unchanged (`documents/sortBy/sortOrder/onSort/onArchive`) —

   (parent owns the asc/desc toggle), DualStatusBadge, VersionHistoryPanel

   typ badges, conditional Dokumentnr/Granskningsdatum rows via renderCard

   `DocumentActionsMenu` shared between the table column and the card
   kebab (`slots.cardActions`). Cell-level `stopPropagation` calls removed
   — the core's interactive-element guard covers them.
