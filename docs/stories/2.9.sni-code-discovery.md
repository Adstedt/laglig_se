# Story 2.9: Create SNI Code-Based Multi-Content Discovery

## Status

Draft

## Story

**As a** visitor,
**I want** to enter my industry code (SNI) and see all relevant legal content,
**so that** I understand my sector's complete compliance landscape.

## Acceptance Criteria

1. SNI discovery page: `/upptack-lagar/bransch`
2. Input field for SNI code (5 digits)
3. SNI code validation
4. Industry starter packs for 15 common sectors
5. Each pack contains: 12-25 SFS laws, 3-8 court cases, 5-12 EU regs/directives
6. Tabbed view: Lagar / Rättsfall / EU-lagstiftning
7. Sortable by relevance, date, category
8. "Lägg till i Min Lista" CTA (requires auth)
9. SEO-optimized industry pages
10. SNI → content mapping in database
11. Mobile-responsive with tab navigation
