# Story 2.8: Implement Cross-Document Navigation System

## Status
Draft

## Story

**As a** user,
**I want** to navigate between related documents (law → cases citing it → EU directive requiring it),
**so that** I understand the complete legal landscape.

## Acceptance Criteria

1. SFS law pages display "Referenced in Court Cases" section
2. Court case pages display "Cited Laws" section
3. EU directive pages display "Swedish Implementation" section
4. SFS law pages display "Implements EU Directive" section
5. Cross-references extracted during ingestion (Stories 2.2, 2.3, 2.4)
6. Manual cross-reference creation interface for authenticated users
7. Bidirectional navigation (A → B means B → A)
8. Cross-reference links show context snippet
9. Cross-reference counts shown
10. Mobile-responsive sections
11. Sample verification
