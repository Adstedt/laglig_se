## FAQ (3-5, formulerade som riktiga sökfrågor)
- **Vad är arbetstidslagen?** — Lag (1982:673) som reglerar hur mycket man får arbeta och hur vila ska förläggas; en ramlag som ofta kompletteras eller ersätts av kollektivavtal.
- **Hur lång dygnsvila kräver arbetstidslagen?** — Minst 11 sammanhängande timmars vila per 24-timmarsperiod, med vissa undantag.
- **Hur mycket övertid är tillåten?** — Förklara gränser för allmän övertid per fyraveckors-/kalendermånadsperiod och per år, samt att avvikelser kan göras i kollektivavtal.
- **Kan arbetstidslagen avtalas bort?** — Ja, delar av lagen kan ersättas genom kollektivavtal så länge EU:s minimiskydd respekteras.
- **Vad händer vid brott mot arbetstidslagen?** — Arbetsmiljöverket utövar tillsyn och kan besluta om sanktionsavgift; beskriv exempel.
