# Innehallsbrief — Kom igang

- **Route:** /kom-igang
- **Cluster / template:** sidor-hub / conversion-landing
- **Sokintention:** transactional (konvertering)
- **Ordmal:** ~400

## Nyckelord

- **Primart:** kom igang med laglista
- **Sekundart:** skapa laglista, testa lagbevakning, prova lagefterlevnad
- **Long-tail:**
  - "skapa laglista pa egen hand"
  - "kom igang med lagefterlevnad snabbt"
  - "testa laglista med organisationsnummer"
  - "borja med lagbevakning for foretag"
  - "starta gratis test lagefterlevnad"
  - "satta upp laglista steg for steg"

## SEO-meta

- **Meta-titel (<=60 tecken):** Kom igang — skapa er laglista pa minuter
- **Meta-beskrivning (~155 tecken):** Kom igang med Laglig.se. Testa med ert organisationsnummer, fa ett forslag pa relevanta lagar och bygg er forsta laglista. Starta er provperiod idag.
- **H1:** Kom igang med Laglig.se

## Produktvinkel (hook)

En kort, fokuserad landningssida med ETT mal: fa besokaren att gora organisationsnummerkollen och starta provperioden. Minimal navigation, ingen artikeltext — bara vagen in.

## Sidstruktur (H2/H3)

1. **MarketingHero** — H1 + en mening om loftet + stor OrgCheck som primar CTA
   - Eyebrow: "Kom igang"
   - Enda primara CTA: "Testa med ert organisationsnummer".
2. **ProcessSteps — Sa kommer ni igang (3 steg)**
   - H3: 1) Ange organisationsnummer → forslag pa relevanta lagar.
   - H3: 2) Bekrafta omraden och bygg er laglista.
   - H3: 3) Aktivera lagbevakning och bjud in kollegor.
3. **ProofBlock — Trygghet att borja**
   - H3: Kort referens/sociala bevis, inget langt resonemang.
4. **OrgCheckCta** — upprepad konverteringsmodul (samma primara CTA)
5. **FaqAccordion** — fa, korta, friktionsavlagsnande fragor
6. **CtaBlock** — sista uppmaning: starta provperiod / boka demo

## FAQ (3-5)

- **Kostar det nagot att komma igang?**
  - Beskriv provperiod/test utan kop; var tydlig och friktionsfri.
- **Vad hander nar jag anger organisationsnumret?**
  - Ni far ett forslag pa lagar och omraden som typiskt gäller er verksamhet — utgangspunkt for laglistan.
- **Hur lang tid tar det att satta upp en laglista?**
  - Minuter for ett forsta utkast; forfining sker lopande. Lank till /funktioner/importera-laglista for befintliga listor.
- **Kan jag bjuda in kollegor?**
  - Ja — lank till /funktioner/ansvar-samarbete.
