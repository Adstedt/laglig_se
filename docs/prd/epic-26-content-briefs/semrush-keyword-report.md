## Strategiska omdirigeringar (datat överstyrde briefen)

1. **egenkontroll:** briefens "egenkontroll miljö" (20/mån) → breda "egenkontroll" (880) med multi-regim-struktur
2. **eu-taxonomin:** briefens primär ~0 volym → "taxonomin" (1 300) arbetas in, H1 behåller disambiguering
3. **sam:** "SAM" (6 600) för tvetydig (KD 45) → fullfrasen primär, SAM som alias
4. **afs / dora / reach / las:** tvetydiga huvudtermer → disambiguerade rubriker ("AFS — Arbetsmiljöverkets föreskrifter", "DORA-förordningen", "REACH-förordningen", "LAS — lagen om anställningsskydd")
5. **iso-45001:** certifieringstermen (3 600/19) större än grundtermen → certifiering i title + tidig process-H2
6. **säkerhetsblad** (1 000/19) utan "data" — folkets stavning bärs jämte den formella på REACH-sidan

## SERP-fynd (phrase_organic)

- **Lagtermer** (avfallsförordningen, arbetstidslagen): SERP = lagtext (riksdagen/lagen.nu) + vägledning + en lagbevakningsaktör (Notisum #6) → **vi är enda aktören med båda halvorna** (fri lagtext i katalogen + explainer) — mönstret replikerat på alla lagförklaringssidor
- **ISO-termer**: SERP = certifieringsbolag + SIS → differentiering via lagkravs-/bindande krav-vinkeln
- **GDPR**: IMY + internationella explainer-sajter → svensk företagsvinkel + katalogkoppling
- **SBA/brandskydd**: räddningstjänsternas vägledningssidor → praktisk hur-gör-man-struktur
