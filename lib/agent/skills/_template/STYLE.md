examples — principle #5). This is where Swedish citation format, phrasing, and
before/after examples live.

- Exempel (svenska): "Arbetsgivaren ska systematiskt undersöka arbetsförhållandena[Källa: SFS 1977:1160, Kap 3, 2 §]."
